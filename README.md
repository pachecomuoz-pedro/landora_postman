# Landora API (Postman Collection)

## Descripción
Colección de Postman para probar los endpoints de la **Landora API**.

## Contenido
- Endpoints de la Landora API
- Variables de entorno configuradas
- Ejemplos de requests y responses

## Requisitos
- [Postman](https://www.postman.com/downloads/) instalado
- Acceso a la Landora API

## Instalación
1. Clona este repositorio
2. Abre Postman
3. Importa la colección desde este proyecto

## Uso
1. Configura las variables de entorno necesarias
2. Selecciona un endpoint
3. Haz clic en "Send" para probar

## Estado
En desarrollo - Actualmente solo contiene endpoints de Landora API
